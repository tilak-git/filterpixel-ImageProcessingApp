**Getting Started:**

1. **Clone Repository:**
    ```bash
    $ git clone https://github.com/tilak-git/filterpixel-ImageProcessingApp.git
    ```
    
2. **Setup backend server**
    ```bash
    $ cd BE
    $ npm i
    $ npm start
    ```
3. **Setup FrontEnd**
    ```bash
    $ cd FE
    $ npm i
    $ npm start
    ```
